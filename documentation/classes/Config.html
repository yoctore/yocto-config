Config Class

Defined in: src/index.js:11

Yocto config controller.
Manage your configuration file (all / common / env & specific file)

Config file has priority. And priority is defined like a php ini system. (Other file).json < all.json < common.json < development.json < stagging.json < production.json

All specific data must be configured on a each correct file.

all.json : contains general data common.json : must containt all common data between each env development.json : must contains development data for development environnement staging.json : must contains stagging data for staging environnement production.json : must contains production data for production environnement
